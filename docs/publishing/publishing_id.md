### Memindahkan Bruno ke pengelola paket baru.

Meskipun kode sumber kami bersifat sumber terbuka dan dapat digunakan oleh siapa saja, kami mohon agar Anda menghubungi kami terlebih dahulu sebelum mempertimbangkan untuk memublikasikannya di pengelola paket yang baru. Sebagai pencipta Bruno, saya memegang hak merek dagang `Bruno` untuk proyek ini dan ingin mengelola distribusinya sendiri. Jika Anda ingin melihat Bruno tersedia di package manager baru, silakan ajukan issue di GitHub."

Meskipun sebagian besar fitur kami gratis dan sumber terbuka (yang mencakup API REST dan GraphQL), kami berupaya mencapai keseimbangan yang harmonis antara prinsip-prinsip sumber terbuka dan keberlanjutan - https://github.com/usebruno/bruno/discussions/269
