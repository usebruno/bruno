### Mempublikasikan Bruno ke Package Manager Baru

Meskipun kode kami bersifat open source dan tersedia untuk digunakan oleh semua orang, kami mohon dengan hormat agar Anda menghubungi kami terlebih dahulu sebelum mempertimbangkan publikasi di package manager baru. Sebagai pencipta Bruno, saya memegang merek dagang Bruno untuk proyek ini dan ingin mengelola distribusinya. Jika Anda ingin melihat Bruno di package manager baru, silakan buat GitHub issue.

Meskipun sebagian besar fitur kami gratis dan open source (yang mencakup REST dan GraphQL API),
kami berusaha untuk mencapai keseimbangan harmonis antara prinsip open-source dan keberlanjutan - https://github.com/usebruno/bruno/discussions/269
